# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-18

### Added

- Initial release.
- `getHolidays`, `isHoliday`, `getHoliday`, and `getNextHoliday` for the DACH region.
- Holiday data for all 16 German states, 9 Austrian Bundesländer, and 26 Swiss cantons.
- Easter and Easter-derived holidays via the Anonymous Gregorian algorithm.
- German, English, French (Romandie), and Italian (Ticino) holiday names.
- Dual ESM/CJS build with bundled TypeScript declarations.
